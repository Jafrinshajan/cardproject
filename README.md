# Job Portal UI

A responsive job portal interface built with React and Vite. This project displays job listings from leading technology companies using reusable React components and a clean user interface.

## 🚀 Features

- Responsive job cards
- Company logos
- Job title, salary, and location
- Full-time/part-time and experience tags
- Reusable React components

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS
- Lucide React

## 📦 Installation

```bash
npm install
npm run dev
```
